What exactly is a favicon?

A favicon is a small square image that represents a website in web browsers. Favicons typically feature a logo, the first letter of the company name or simply an image that reflects the specifics of the website it represents.



Below are just a few examples of some favicons you may be familiar with (with respectable links).



How are favicons created?

You can design a favicon image using an editor like Fireworks, Photoshop, Corel Paint, or a free, open-source alternative like GIMP.
In this class we creatd a favicon with Studio code.


How did I create the favicon for this site?

For this site I created my favicon with studio code. I set my image to 20px by 20px. I decided to go with the initial letter of my first name "C". I found the software somewhat hard to use so I decided to go with something simple.
